# 👥 Swarm-Baremetal (L'Intelligence Collective / Phéromones)

## Rôle Biologique
Un Operating Organism ne vit pas forcément seul. `swarm-baremetal` est l'organe qui permet à plusieurs organismes de communiquer entre eux de manière P2P (Machine to Machine), comme des fourmis échangeant des phéromones.

## Connectivité
- **Partage d'Immunité** : Si l'organisme A découvre un nouveau virus et forge un anticorps (`bot-baremetal`), le Swarm diffuse cet anticorps à l'organisme B. L'immunité est collective (Herd Immunity).
- **Consensus** : Travaille avec `united-baremetal` pour prendre des décisions distribuées si le cluster est attaqué.
