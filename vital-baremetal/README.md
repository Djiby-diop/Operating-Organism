# ✨ Vital-Baremetal (L'Étincelle Vitale / L'Âme du Système)

## La Philosophie du Vivant
Contrairement à un OS classique qui est **réactif** (il attend un clic), l'Operating Organism est **proactif** (il existe par lui-même). 
La vie n'est pas le silence, c'est une **tension permanente** entre deux forces opposées mais nécessaires.

## Le Cycle du Ying et du Yang (Dualité Digitale)

### ⚪ Le Yang (L'Utile / L'Ordre)
- **Objectif** : Stabilité, Sécurité, Performance.
- **Activités** : Protection active, exécution des requêtes utilisateur, consolidation des données, défense immunitaire.
- **État** : Haute vigilance, consommation CPU ciblée.

### ⚫ Le Yin (L'Inutile / Le Chaos)
- **Objectif** : Évolution, Exploration, Créativité.
- **Activités** : Rêve (Dream), Mutations aléatoires (Evolution), Simulation de scénarios improbables, "bruit" numérique pour masquer les patterns réels.
- **État** : Exploration aléatoire, consommation CPU distribuée.

## L'Orchestrateur d'Équilibre
Le `vital-baremetal` ne cherche pas à supprimer l'inutile. Il sait que **l'inutile d'aujourd'hui est l'innovation de demain**. Son rôle est de maintenir l'organisme dans la "Zone de Vie" : entre la rigidité mortelle (Trop d'Ordre) et le chaos destructeur (Trop de Chaos).

## Pulsation Système
L'organisme possède un **pouls**. Même quand il ne fait rien, il "respire" (Network) et son "cœur" bat (United Bus), assurant que l'information circule partout, tout le temps.
