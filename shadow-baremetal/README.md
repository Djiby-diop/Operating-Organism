# 🌑 Shadow-Baremetal (L'Instinct de Survie Sombre)

## Rôle Biologique
Si l'organisme est chassé ou compromis par une menace de niveau `SURVIVAL`, il doit devenir invisible et indétectable pour survivre.

## Le Côté Sombre
- **Furtivité (Chameleon)** : Hook des tables système pour masquer la présence du Kernel ou des Agents Immunitaires aux outils d'analyse externes (Rootkit bienveillant).
- **Anti-Forensics** : Capacité à s'auto-effacer de la mémoire, chiffrer l'espace disque instantanément, ou simuler la mort de l'organisme (Fake Halt).
- Intervient uniquement quand la mort de l'organisme est imminente.
